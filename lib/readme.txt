bc-x jars are BouncyCastle (http://www.bouncycastle.org), License is BC (BSD like).
These jars are the cryptographic foundation in EJBCA.
Current version used is 1.45.

Ldap.jar is from OpenLDAP and used for some parsing of distinguished named, current version is from 2009-10-07. License can be found at http://www.openldap.org/software/release/license.html.

Log4j-1.2.16.jar is from apache, License is Apache.
The file is used for logging throughout EJBCA.
Version is 1.2.16.

cert-cvc.jar is a library for handling Card Verifiable Certificates, developed
by Keijo Kurkinen, and contributed to EJBCA. License is LGPL.
Version is 1.2.13.
cert-cvc.jar is built from the sub project cert-cvc of EJBCA.

ehcache-core-2.3.1.jar is used for caching. License is Apache 2.0.

jackson-all-1.7.0.jar is used for JSON processing. License is Apache 2.0.
